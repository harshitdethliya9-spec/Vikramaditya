# Vikram Aditya Palace & Marriage Garden Web App ✦

Welcome to the official web application repository for **Vikram Aditya Pallace and Marriage Garden** (Katni - Jabalpur Road Area, MP, India). 

This is a premium, high-fidelity, and fully responsive single-page web application designed with a **royal navy & gold theme**. It showcases the venue's facilities, rooms, and provides an interactive availability calendar allowing direct WhatsApp booking enquiries.

---

## ✨ Features & Highlights

- 📸 **Luxury Image Slider**: A smooth, auto-playing slideshow featuring actual views of the hotel building under sunset, lush green lawns, and architectural room layouts.
- 🏨 **Facilities Showcase**: Details of venue capacity, including:
  - 4 Premium AC Rooms & 5 Non-AC Rooms (with attached washrooms).
  - 1 Grand Banquet Hall (with separate toilets).
  - Lush Green Marriage Lawn.
  - Secured parking space for 60+ cars.
  - Onsite booking office (8 AM - 10 PM) and 24/7 function support.
- 📅 **12-Month Availability Calendar**: An interactive calendar grid color-coded for quick verification:
  - <span style="color:#10B981; font-weight:bold;">Green (Available)</span>: Selectable dates free for booking.
  - <span style="color:#EF4444; font-weight:bold;">Red (Booked)</span>: Reserved dates crossed out.
- 💬 **WhatsApp Booking API**: Clicking any available date opens an elegant inquiry modal. Submitting the form formats a polite, details-filled Hinglish message and instantly opens a WhatsApp chat window with the venue manager (+91 9752671437).
- 📍 **One-Click Location Sync**: Integrates a custom royal location card linking directly to [Google Maps Location Guide](https://maps.app.goo.gl/9FiNPXyAaGek7Zek8?g_st=aw).
- ⚡ **Zero-Dependency execution**: Pure Vanilla HTML5, CSS3, and ES5 JavaScript. Runs perfectly offline directly from the file system without triggering browser CORS blocks.

## 🚀 How to Run the Project Locally

No installation or command-line compilation is required!

1. Clone this repository to your machine:
   ```bash
   git clone https://github.com/harshitdethliya9-spec/Vikramaditya.git
   ```
2. Open the project folder and double-click [index.html](index.html) to run the website instantly in Google Chrome, Microsoft Edge, Safari, or Firefox.
